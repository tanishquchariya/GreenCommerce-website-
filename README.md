# 🛒 GreenCommerce – E-Commerce Website

GreenCommerce is a modern **React-based e-commerce web application** that allows users to browse products, manage their shopping cart, and place orders through a smooth and responsive interface.

The platform is designed to simulate a real-world online shopping experience with **secure authentication, dynamic product catalog, and scalable front-end architecture**.

# 📌 Key Features

## 👤 User Authentication

* Secure **Login and Signup**
* Email and password validation
* Authentication flow for protected routes

## 🛍️ Product Catalog

* Dynamic product listing
* Product details with description and price
* Clean UI for product exploration

## 🛒 Shopping Cart

* Add products to cart
* Remove products from cart
* Update product quantities
* Real-time cart state management using Redux

## 💳 Checkout & Order Processing

* Order summary before purchase
* Payment gateway simulation
* Order confirmation interface

## 📦 Order Tracking

* Users can track order status
* Provides shipping and delivery updates

## 🎨 Responsive UI

* Mobile-friendly design
* Clean and modern interface using **Chakra UI**
* Expandable and collapsible sidebar navigation
